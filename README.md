# AI UMKM Co-Pilot

AI UMKM Co-Pilot adalah aplikasi manajemen keuangan dan bisnis sederhana untuk UMKM kecil.

Aplikasi ini membantu pengguna mencatat pemasukan, pengeluaran, produk, stok barang, dan mendapatkan insight bisnis melalui AI assistant.

## Target Pengguna

- Warung kecil
- Toko rumahan
- Online seller
- Reseller
- Usaha makanan dan minuman
- Jasa kecil

## Masalah yang Diselesaikan

Banyak UMKM belum memiliki pencatatan keuangan yang rapi, tidak tahu laba bersih secara akurat, stok sering tercecer, dan sulit mengambil keputusan bisnis berdasarkan data.

## Fitur MVP

- Login dan register
- Profil usaha
- Catat pemasukan
- Catat pengeluaran
- Kategori transaksi
- Dashboard bisnis
- Laporan laba rugi sederhana
- Manajemen produk
- Manajemen stok
- AI assistant untuk insight bisnis

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- Vercel
- AI API

Project sedang dalam tahap perencanaan dan pengembangan MVP.
