# Roadmap — AI UMKM Co-Pilot

## 1. Tujuan Roadmap

Dokumen ini menjelaskan tahapan pengembangan AI UMKM Co-Pilot dari tahap dokumentasi awal sampai MVP beta.

Roadmap digunakan agar pengembangan berjalan rapi, terukur, dan tidak keluar dari scope MVP.

---

## 2. Prinsip Roadmap

Pengembangan aplikasi mengikuti prinsip berikut:

1. Dokumentasi dibuat sebelum coding.
2. Fitur dikerjakan bertahap.
3. Setiap fitur memiliki branch dan commit sendiri.
4. Scope MVP dijaga agar tidak melebar.
5. Fitur utama harus selesai sebelum fitur tambahan.
6. Aplikasi harus mobile-first.
7. AI hanya digunakan setelah data bisnis dasar tersedia.

---
